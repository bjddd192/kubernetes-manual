# k9s

K9s 是 一款管理员们喜欢的 “单一屏幕” 实用程序，K9s 提供了一个基于 curses 的全屏终端 UI，可与您的 Kubernetes 集群进行交互。 

K9s 会以特定时间间隔（默认为2秒）监控Kubernetes资源，并允许我查看自己集群中的内容。它可以一目了然地为你提供了运行中 Pod、日志和部署的可视化视图，以及对 Shell 的快速访问。
